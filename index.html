What is Abyss03 Nexus?

A maintained fork of the original Abyss Web Server tailored for Dust 514 preservation and local testing. Routes HTTPS via stunnel, captures the first real request/response, and lets you iterate quickly with a replay harness.

Check Health or hit /dust/auth/login if stubbed

Quick Setup

  1. Run stunnel on 127.0.0.1:443 → Abyss03 Nexus HTTP port
  2. Start Abyss03 Nexus and trigger the client handshake
  3. Inspect logs, save capture, then replay to iterate

Replay Harness

Use the Python CLI abyss03 --replay captures/first_req.txt to push real requests through stunnel and validate responses, without restarting the game client.
